# 📊 Array Programs

This folder contains C++ implementations for various array and matrix manipulation problems. These programs cover basic operations, searching, sorting, and matrix mathematics.

| SN  | Problem Name              | File Name                                                                                                              | Description                                                                    |

| 2   | **Read & Display**        | [array_read_display.cpp](file:///media/sandipghimire/ClydeSan/College/C++/ArraysPrograms/array_read_display.cpp)       | Basic demonstration of reading array elements from user and displaying them.   |

## 🛠️ How to Compile and Run

You can compile these programs using any C++ compiler (like g++).

```bash
g++ [filename].cpp -o [output_name]
./[output_name]
```
